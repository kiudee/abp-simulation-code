Aggregated Bandits Simulation

Simulation environment for the aggregated bandits problem.

Download this project as a .zip file Download this project as a tar.gz file

Required software

The simulation environment was implemented in Python 2. The following packages were used:

  • Numpy 1.9.1
  • Pandas 0.15.2
  • SciPy 0.15.1
  • Seaborn 0.5.1
  • IPython 2.3.1

Running the experiments

The simulation environments and the algorithms are contained in aggregated_bandits.py. The code to run the experiments can found in the IPython notebook MaxPeriodSolver.ipynb.
